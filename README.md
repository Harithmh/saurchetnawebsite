# Saur Chetna Evam Urja Vigyan Shodh Sansthan Website

This repository contains the public website for Saur Chetna Evam Urja Vigyan Shodh Sansthan, Bikaner.

## Files

- `SchoolWebsite.html` - Main website page
- `index.html` - GitHub Pages entry file that opens the main site
- `logo-saur-chetna.svg` - Custom logo and favicon
- `Gallery/` - Website images
- `.nojekyll` - Ensures GitHub Pages serves files directly

## Features

- Responsive public website layout
- Accessibility toolbar
- Hindi translation toggle
- Read-aloud support
- Image gallery with lightbox
- Contact form opening email compose window
- SEO metadata and structured data

## Run Locally

Open `SchoolWebsite.html` directly in a browser, or use a simple local server.

## Publish On GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch.
7. Select the `/ (root)` folder.
8. Save the settings.
9. GitHub will publish the website using `index.html`.
